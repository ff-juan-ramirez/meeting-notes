# Milestone v1.2: Named Recordings

**Status:** ✅ SHIPPED 2026-03-29
**Phases:** 02-07
**Total Plans:** 6

## Overview

Allow `meet record` to accept an optional name that propagates to all output files, metadata, `meet list` title display, Notion page titles, and runtime title overrides. Milestone expanded from original 4-phase scope (02-05) to include Session ID column (06) and `--title` flag (07).

## Phases

**Goal:** Implement `slugify()` pure function and `get_recording_path_with_slug()` in `core/storage.py` using TDD. Zero new dependencies.
**Requirements covered:** SLUG-01, SLUG-02, RECORD-05
**Status:** ✅ Complete (2026-03-28)
**Plans:** 1/1

Plans:
- [x] 02-01-PLAN.md — TDD slugify() and get_recording_path_with_slug() pure functions

**Details:**
- `slugify(text)`: NFKD normalization, ASCII-safe, lowercase, colon→hyphen, 80-char max, `"untitled"` fallback

## Milestone Summary

**Decimal Phases:** None

**Key Decisions:**
- Optional positional `[NAME]` argument over `--name` flag — more natural for recording naming
- `slugify()` as pure function in `core/storage.py` — testable without side effects
- `slugify()` truncates at 80 chars with trailing hyphen strip — handles edge cases cleanly
- Falsy check (`if recording_name:`) used uniformly across phases 04, 05, 07 — None/empty/missing all fall through
- Session ID column added with no max_width — untruncated by design for `--session` round-trip reliability
- `notion_title` local variable in phase 07 — avoids shadowing Click parameter name
- `--title` not persisted to metadata — runtime override only, preserves clean separation

**Issues Resolved:**
- Phase 05 UAT revealed transcribe command was overwriting `recording_name` in metadata — fixed with `preserve_recording_name` guard (debug session between phases 05 and 07)
- pyannote/ffmpeg warning during doctor suppressed (post-milestone fix)

**Issues Deferred:**
- SESSRES-01: human-readable name as `--session` argument (slug-prefix glob match) — deferred, exact stem works
- LIST-03: Dedicated "Name" column distinct from derived title — deferred, low ROI

**Technical Debt Incurred:**
- REQUIREMENTS.md only covered original 10 requirements (phases 02-05); SESSID-* and TITLE-* requirements tracked in PROJECT.md but not added to REQUIREMENTS.md at definition time
