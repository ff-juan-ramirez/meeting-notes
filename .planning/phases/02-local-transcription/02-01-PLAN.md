    Create the file with these functions:

    1. `transcribe_audio(wav_path: Path, config: Config) -> str` — per D-12, calls `mlx_whisper.transcribe(str(wav_path), path_or_hf_repo="mlx-community/whisper-large-v3-turbo", **decode_opts)`. CRITICAL per Pitfall 1: when `config.whisper.language is None`, do NOT include `language` key in decode_opts at all. Only add `decode_opts["language"] = config.whisper.language` when it is not None. Returns `result["text"]`.

    2. `estimate_wav_duration_seconds(wav_path: Path) -> float` — Calculates duration from file size: `BYTES_PER_SECOND = 16000 * 2` (32000), `WAV_HEADER_BYTES = 44`. Formula: `max(0, wav_path.stat().st_size - WAV_HEADER_BYTES) / BYTES_PER_SECOND`. No ffprobe dependency.

    3. `run_with_spinner(fn, message: str) -> Any` — Runs `fn()` in a `threading.Thread(target=worker, daemon=True)`. Main thread shows a Rich `Live` display with elapsed time `[Ns]` that updates via `time.sleep(0.1)` loop. When thread finishes, joins it and returns the result. If worker raised an exception, re-raises it in the main thread. Uses `rich.live.Live`, `rich.text.Text`, `rich.console.Console`.

    Constants at module level:
    ```python
    BYTES_PER_SECOND = 16000 * 2  # 32000 — 16kHz mono s16le
    WAV_HEADER_BYTES = 44
    WARN_DURATION_SECONDS = 90 * 60  # 5400
    WARN_WORD_COUNT = 50
    MODEL_REPO = "mlx-community/whisper-large-v3-turbo"
    ```

    **Step 1: Create CLI command (meeting_notes/cli/commands/transcribe.py)**

    Follow the exact structure of `record.py`: module-level `console = Console()`, `@click.command()` decorator, imports from core and services.

    The command signature:
    ```python
    @click.command()
    @click.option("--session", default=None, help="WAV filename stem (e.g. 20260322-143000-abc12345)")
    def transcribe(session: str | None) -> None:
    ```

    Command logic flow:
    1. Get paths: `recordings_dir = get_data_dir() / "recordings"`, `transcripts_dir = get_data_dir() / "transcripts"`, `metadata_dir = get_data_dir() / "metadata"`
    2. Resolve WAV file:
       - If `session is None`: call `resolve_latest_wav(recordings_dir)` — per D-01
       - If `session is not None`: call `resolve_wav_by_stem(recordings_dir, session)` — per D-02
       - On `FileNotFoundError`: print `[red]Error:[/red] {exception message}` and `sys.exit(1)`
    3. Extract stem: `stem = wav_path.stem`
    4. Estimate duration: `duration = estimate_wav_duration_seconds(wav_path)`
       - If `duration > WARN_DURATION_SECONDS`: print `[yellow]Warning:[/yellow] Recording is over 90 minutes. Transcription may cause memory pressure on Apple Silicon.` — per D-14
    5. Load config: `config = Config.load(get_config_dir() / "config.json")`
    6. Check model cache: if `Path.home() / ".cache" / "huggingface" / "hub" / "models--mlx-community--whisper-large-v3-turbo"` does not exist, use `run_with_spinner(lambda: transcribe_audio(wav_path, config), "Downloading model and transcribing...")` — per D-07. Otherwise use `run_with_spinner(lambda: transcribe_audio(wav_path, config), "Transcribing...")` — per D-16.
    7. Post-transcription: `text = result.strip()`
       - `word_count = len(text.split())`
       - If `word_count < WARN_WORD_COUNT`: print `[yellow]Warning:[/yellow] Transcript may be empty — check audio routing ({word_count} words)` — per D-13/TRANS-04
    8. Save transcript: `transcript_path = transcripts_dir / f"{stem}.txt"` — per D-05, D-06
       - `transcript_path.write_text(text)` — overwrites silently per D-03
    9. Save metadata: `metadata_path = metadata_dir / f"{stem}.json"` — per D-10
       - Fields per D-11:
         ```python
         metadata = {
             "wav_path": str(wav_path.resolve()),
             "transcript_path": str(transcript_path.resolve()),
             "transcribed_at": datetime.now(timezone.utc).isoformat(),
             "word_count": word_count,
             "whisper_model": "mlx-community/whisper-large-v3-turbo",
         }
         ```
       - `write_state(metadata_path, metadata)` — atomic write per existing pattern
    10. Print success: `console.print(f"[green]Transcription complete[/green] ({word_count} words)")` then `console.print(f"Session: {stem}")` — per D-04 and CONTEXT.md specifics

    Helper functions (at module level, NOT inside the command):
    ```python
    def resolve_latest_wav(recordings_dir: Path) -> Path:
        wavs = sorted(recordings_dir.glob("*.wav"), key=lambda p: p.stat().st_mtime)
        if not wavs:
            raise FileNotFoundError("No recordings found in recordings directory.")
        return wavs[-1]

    def resolve_wav_by_stem(recordings_dir: Path, stem: str) -> Path:
        candidate = recordings_dir / f"{stem}.wav"
        if not candidate.exists():
            raise FileNotFoundError(f"No recording found for session: {stem}")
        return candidate
    ```
