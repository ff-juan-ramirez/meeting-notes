       - Continue until all text consumed

       - Other non-empty lines become paragraph blocks
       - Each line's text is passed through _split_text; each chunk becomes a separate block of the same type
       - Block shape: `{"object": "block", "type": block_type, block_type: {"rich_text": _rich_text(chunk)}}`

       - On non-429 APIResponseError or exhausted retries, raise
       - On success, return result
