```python
class SessionRowWidget(QFrame):
    def __init__(self, session_id: str, title: str, subtitle: str,
                 status: str, parent=None) -> None:
```

- Layout: `QHBoxLayout` containing:
  - Left `QVBoxLayout`:

    - "transcribed" -> accent `#0A84FF`
    - "not-transcribed" -> border `#E5E5EA`
- `self._session_id = session_id` stored as public-readable attribute
- `sizeHint()` returns `QSize(0, 56)` — preferred height 56px per UI-SPEC
- Do NOT import anything from services or CLI modules — widget is pure presentation
